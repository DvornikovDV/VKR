# VKR SCADA System

Распределенная SCADA-система, организованная по принципу монорепозитория. Включает в себя веб-конструктор мнемосхем, панель мониторинга и серверную часть.

## Структура проекта

### Клиентские модули (Client Modules)

* `/client/` — React SPA shell. Единая точка входа, предоставляющая публичную зону, панель администратора (Admin Hub) и панель пользователя (User Hub). Встраивает `/constructor` и `/dashboard` через компоненты-обертки.
* `/constructor/` — Клиентский модуль визуального реактора (холста) для проектирования дашбордов, размещения виджетов и настройки привязки.
* `/dashboard/` — Клиентский модуль операторского интерфейса для рендеринга готовых мнемосхем и оперативных данных в реальном времени.

### Серверные модули (Server Modules)

* `/cloud_server/` — Облачный бэкенд для агрегации данных, глобальной синхронизации состояния и хранения конфигураций мнемосхем.
* `/edge_server/` — Пограничный контроллер (локальный сервер) для управления и общения с реальным (физическим) оборудованием по промышленным протоколам.
