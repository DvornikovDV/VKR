# VKR SCADA System

Распределенная SCADA-система, организованная по принципу монорепозитория. Включает в себя веб-конструктор мнемосхем, панель мониторинга и серверную часть.

## Структура проекта

* `/constructor/` — Клиентский модуль веб-конструктора мнемосхем. Позволяет визуально проектировать дашборды, размещать виджеты и настраивать привязку к физическим устройствам.
* `/dashboard/` — Клиентский модуль для отображения и взаимодействия с готовыми мнемосхемами в режиме "только чтение" (операторский интерфейс).
* `/cloud_server/` — Облачный бэкенд для агрегации данных, глобальной синхронизации и хранения конфигураций.
* `/edge_server/` — Пограничный контроллер (локальный сервер) для управления и общения с реальным оборудованием по промышленным протоколам.

## Регламент разработки
Локально проект содержит архитектурную документацию (`АРХИТЕКТУРА_ПРОЕКТА.md`, `scada-architecture.md` и др.) и AI-агентов (`.agent/`), которые игнорируются системой контроля версий (Git). В репозиторий уходит только исходный код самих модулей и базовые конфигурации системы.
