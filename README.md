# VKR SCADA System

Распределенная SCADA-система, организованная по принципу монорепозитория. Включает в себя веб-конструктор мнемосхем, панель мониторинга и серверную часть.

## Структура проекта

- **Клиентские модули**:
  - `/client`: Оболочка React SPA — единая точка входа, обеспечивающая доступ к публичной зоне, панели администратора (Admin Hub) и панели пользователя (User Hub). Интегрирует `/constructor` и `/dashboard` через компоненты-обертки.
  - `/constructor`: Визуальный редактор мнемосхем с загрузкой фоновых изображений, созданием связей и размещением виджетов.
  - `/dashboard`: Интерфейс оператора для отображения мнемосхем и данных мониторинга на них в реальном времени.
- **Серверные модули**:
  - `/cloud_server`: Облачный бэкенд, отвечающий за агрегацию данных, глобальную синхронизацию состояний и хранение конфигураций мнемосхем.
  - `/edge_server`: Локальный пограничный контроллер (Edge), взаимодействующий с физическим оборудованием по промышленным протоколам.
