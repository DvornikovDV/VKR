# Конструктор мнемосхем (Constructor)

Визуальный HTML5/Canvas веб-конструктор. Является одной из подсистем VKR SCADA. 

## Назначение
Предоставляет графический интерфейс для:
1. Загрузки и позиционирования фонового изображения (мнемосхемы предприятия).
2. Создания и трассировки соединений между изображениями.
3. Расстановки интерактивных виджетов (датчики, переключатели, табло).
4. Привязки виджетов к конкретным аппаратным ID оборудования.
5. Экспорта итогового файла формата JSON, который впоследствии понимает модуль `dashboard`.

## Архитектура и стек

* **Стек**: Vanilla JS, Konva.js, Vanilla CSS.
* **Паттерн**: Coordinator / Mediator. Центральным узлом связи между всеми модулями является `ui-controller.js`. Менеджеры общаются друг с другом только через генерацию событий (отправку callback-ов Главному контроллеру).

Детальное устройство и файлы для этого модуля описаны в локальном `FILE_MAP.md` данного каталога.
